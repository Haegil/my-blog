# LEVEL-1

[현재 발견되는 개선사항들]

1. 새 글을 작성하기 위해 에디터에 진입하게 되면, 현재 markdown의 문법을 인식하고 변환해주며 그 결과를 실시간으로 볼 수 있음.
![alt](./pdf/screencapture-localhost-5173-write-2026-05-22-14_33_34.pdf)
하지만 위의 이미지와 같이 전체화면이 아닌 좌우의 공간이 많이 비어있음.

2. 메인 페이지의 최신 기록들을 간략화해서 보여주는 카드를 변경하고 싶음.
![alt](./pdf/screencapture-localhost-5173-2026-05-22-14_36_36.pdf)
위 사진을 보면, 제목과 컨텐츠를 모두 보여주고 있음.

3. 메인 페이지에서 중앙 상단에 있는 검색창 기능을 이용할 때, 태그 검색에 대해 사용법이 조금 모호함.

[수정 희망 방향]

1. 글을 작성하는 마크다운 에디터는 왼쪽으로 치우쳐져 있지 않게 위의 다른 입력창과 동일한 너비를 갖게 하고, 컨텐츠를 마크다운으로 미리보기 할 수 있는 토글 버튼 혹은 모달창을 지원했으면 함.
토글 방식으로 할 경우, 에디터의 우측 상단에 '미리보기'라는 버튼을 만들고 그것을 클릭했을때, 에디터가 읽기 전용으로 바뀌면서 마크다운 문법이 적용된 결과를 볼 수 있게끔 만들었으면 함. 별도 애니메이션은 필요없음
모달창으로 할 경우, 이 또한 미리보기 버튼을 에디터의 우측 상단에 만들고 그것을 클릭했을때, 모달창이 떠서 마크다운 문법이 적용된 결과를 볼 수 있게끔 만들었으면 함. 별도 애니메이션은 필요없음
이 두 가지 방식 중 어떤 것이 더 좋을지 판단해서 제안해주고 그에 따라 수정해줘.

2. 메인 페이지의 최신 기록들을 간략화해서 보여주는 카드를 변경하고 싶음.
우선, 컨텐츠는 렌더링 되지 않았으면 함. 대신 카드의 카테고리는 현행대로 진행해도 괜찮음.
대신 한 low에 하나의 카드만 표시되었으면 함. 그리고 최근 기록들을 표시하는 컴포넌트의 우측 상단에 '전체글보기' 버튼을 만들고 전체글 조회가 되는 페이지를 따로 만들었으면 함.
MemoStack Stats도 메인페이지에서 빼고 전체글보기의 main쪽 상단에 위치했으면 함.

3. 메인 페이지에서 제목과 본문 검색은 잘 구현되어있음. 하지만 태그 검색은 사용법이 조금 모호해서 별도의 시그니처를 달고 검색어를 입력하면 태그검색으로 바뀌도록 바꾸고 싶음. 예를들어 '#'를 검색어 앞에 붙이면 태그검색으로 인식되어서 해당 태그를 가진 모든 글이 나오게끔 하고 싶음.
또, 검색으로 글을 조회했을때, 한번에 모든 글이 조회되지 않고 페이징 처리를 했으면 좋겠음.
