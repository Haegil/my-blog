# MemoStack 와이어프레임

## 공통 레이아웃

```text
로고      검색 입력창                           아이콘 액션
------------------------------------------------------------
페이지 콘텐츠
------------------------------------------------------------
푸터
```

## 홈

```text
최신 게시글                                      전체 글 보기
짧은 설명

작성일
게시글 제목
#tag #tag

작성일
게시글 제목
#tag
```

## 에디터

```text
제목 입력
태그 입력

툴바                                          미리보기
------------------------------------------------------------
마크다운 입력 또는 렌더링된 미리보기
------------------------------------------------------------
글자 수                                       저장
```

## 태그

데스크톱:

```text
태그 목록             선택한 태그의 게시글
#react       3        게시글 카드
#express     2        게시글 카드
#postgres    1
```

모바일:

```text
태그 칩을 2열 그리드로 압축 표시
선택한 태그의 게시글
```
