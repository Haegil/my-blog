# 마크다운 에디터 기능 정리

MemoStack의 에디터는 `.md` 파일을 편집하는 경험에 가깝게 동작하는 것을 목표로 한다.

## 입력

- 제목 입력
- 쉼표로 구분한 태그 입력
- 본문 마크다운 입력
- 작성 중 로컬 임시 저장
- 글자 수 표시

## 문법 지원

본문 미리보기는 `react-markdown`과 `remark-gfm`을 사용한다.

지원 범위:

- 제목: `#`, `##`, `###`
- 굵게: `**text**`
- 기울임: `*text*`
- 인라인 코드: `` `code` ``
- 코드 블록: fenced code block
- 인용문: `> text`
- 링크: `[text](url)`
- 순서 없는 목록
- 순서 있는 목록
- GFM 테이블
- 가로선

## 실시간 문법 강조

편집 화면은 실제 폰트 크기와 줄 높이를 유지하면서 문법 요소에 색상만 입힌다. 커서 위치와 표시 텍스트 간격이 어긋나지 않도록 제목 문법도 큰 글꼴로 렌더링하지 않는다.

강조 대상:

- 헤딩 기호
- 굵게/기울임
- 인라인 코드
- 코드 블록
- 인용문
- 목록 마커
- 링크 텍스트와 URL
- 해시태그
- 테이블 파이프와 구분선

## 미리보기

미리보기 모드는 본문 영역 안에서 렌더링된 마크다운 결과를 보여준다. 코드 블록에는 `highlight.js` 문법 강조가 적용된다.
