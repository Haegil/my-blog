# MemoStack PRD

## 제품 목표

MemoStack은 개인 기술 글쓰기와 학습 기록을 위한 블로그 애플리케이션이다. 빠른 마크다운 작성, 태그 기반 탐색, 차분한 읽기 경험에 집중한다.

## 주요 사용자

- 관리자 작성자: 게시글을 작성, 수정, 삭제하고 태그로 정리한다.
- 방문자 독자: 최신 글을 읽고, 검색하고, 태그별로 게시글을 탐색한다.

## 핵심 요구사항

- 홈 화면에서 최신 게시글을 보여준다.
- 전체 게시글 페이지와 페이지네이션을 제공한다.
- 마크다운 작성, 문법 강조, 미리보기를 지원한다.
- 게시글 작성/수정 폼에서 태그를 입력하고 생성할 수 있다.
- 태그 목록과 태그별 게시글 탐색을 제공한다.
- 제목/본문 검색과 `#태그` 검색을 지원한다.
- 홈, 전체 글, 검색 결과, 태그별 목록에서 표시 개수와 작성일 범위 필터를 제공한다.
- 게시글 작성, 수정, 삭제는 관리자 세션으로 보호한다.
- 앱 마운트 시 세션을 확인해 만료된 로그인 상태가 UI에 즉시 반영되도록 한다.

## 운영 참고사항

- 운영 환경의 관리자 계정 정보는 환경변수로 제공해야 한다.
- 운영 환경에서 기본값인 `admin/admin1234`에 의존하면 안 된다.
- 프론트엔드와 백엔드가 서로 다른 도메인에 배포될 경우 세션 쿠키의 cross-site 설정을 정확히 맞춰야 한다.
